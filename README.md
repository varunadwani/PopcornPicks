# PopcornPicks — Movie Finder App

**"Tired of scrolling endlessly to find something to watch? Search, explore, and pick your next movie in seconds with PopcornPicks."**

PopcornPicks is a web-based movie discovery app built to make finding movies simple and fast. Whether you are looking for a recent release, a highly rated film, or just exploring options for movie night, PopcornPicks gives you everything in one place. With search, filtering, sorting, and a personalized favorites system, the app turns a frustrating search into a smooth experience.

---

## The Problem

Most people spend more time deciding what to watch than actually watching something. You search on one platform, check ratings on another, and still end up confused. There is no simple place where you can quickly search, compare, and shortlist movies without distractions.

PopcornPicks solves this by bringing essential movie information together in a clean and focused interface.

---

## What It Does

**Movie Search**  
Instantly search for any movie by name and get relevant results in real time.

**Movie Details**  
View key information including poster, title, release year, and rating in a structured format.

**Filtering Options**  
Narrow down results based on release year and type to find exactly what you are looking for.

**Sorting System**  
Sort movies based on rating, release date, or popularity to make better choices.

**Favorites**  
Save movies you like using localStorage and access them anytime.

**Dark Mode**  
Switch between light and dark themes for a better viewing experience.

---

## API Details

This project uses the TMDB API to fetch movie data.

| API Endpoint     | Purpose in Project                           |
|-----------------|---------------------------------------------|
| /search/movie   | Search movies based on user input           |
| /discover/movie | Filter and sort movies by various criteria  |
| /movie/{id}     | Fetch detailed information of a movie       |
| image base path | Display movie posters and images            |

---

## Tech Stack

Built using core web technologies without heavy frameworks to keep the project simple and efficient.

- HTML for structure  
- CSS for styling  
- JavaScript for functionality  
- TMDB API for movie data  
- Browser localStorage for saving favorites
